# StreetPaws

Street Paws is a community-driven React application designed to bridge the gap between animal lovers and rescue organizations in Pakistan. The platform allows users to report injured or distressed street animals, access essential safety guidelines for handling rescues, and quickly find verified contact information for local NGOs and shelters. By centralizing these resources, Street Paws empowers citizens to take immediate, safe, and effective action to save animal lives.

## Features

* **Rescue Dashboard:** Track rescue reports and community impact.
* **NGO Directory:** A verified list of animal welfare organizations across major cities in Pakistan (Karachi, Lahore, Islamabad, Peshawar).
* **Safety Guidelines:** Interactive "Safety Tips" to help users approach injured animals without causing further harm.
* **Responsive UI:** A modern, "frosted-glass" design with a subtle animal-themed background for a professional look.
* **Reporting System:** Clear steps for users to spot, report, and alert the community about animals in need.

##  Tech Stack

* **Frontend:** React.js
* **Routing:** React Router DOM
* **Styling:** Custom CSS with CSS Variables for theme consistency
* **Icons:** Custom image icons for a professional UI
* **Backend** Firebase Authentication & Firestore database
